Interface (``sparscit.pl.int``)
================================

Simple interface for common plot types. These are lower-level helpers
used by the main plotting module.

- :func:`src.plotting.int._bar.barplot` — Bar plot
- :func:`src.plotting.int._hist.histogram` — Histogram
- :func:`src.plotting.int._heatmap.heatmap` — Heatmap
- :func:`src.plotting.int._violin.violin` — Violin plot
- :func:`src.plotting.int._violin.boxplot` — Box plot
- :func:`src.plotting.int._scatter.scatter` — Scatter plot
