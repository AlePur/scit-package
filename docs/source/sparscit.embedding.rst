Embedding (``sparscit.em``)
============================

Dimensionality reduction and embedding methods for single-cell data.
Includes PCA, spectral embedding, UMAP, dendrogram, and landscape embedding.

- :func:`sparscit.embedding._pca.pca` — Principal Component Analysis
- :func:`sparscit.embedding._spectral.spectral` — Spectral embedding for sparse data
- :func:`sparscit.embedding._spectral.multiview_spectral` — Multi-view spectral embedding
- :func:`sparscit.embedding._umap.umap` — UMAP embedding
- :func:`sparscit.embedding._dendrogram.dendrogram` — Compute dendrogram
- :func:`sparscit.embedding._landscape.landscape` — Create landscape embedding from neighborhood clusters
